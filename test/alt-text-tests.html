Alt-Text Tests

Alt-text should describe the intent and or context of the image not random details. If the image adds no info, it can be marked decorative.

Image Alt Texts

alt="{description}"

Image with alt-text content. Note title=.. to add additional info (tooltip does not equal alt-text)

alt=""

Image without alt-text content, marked as decorative using alt=""

aria-hidden="true"

Image without alt-text content, marked as decorative (not exposed to a11y API) using aria-hidden=true

role="none"

Image without alt-text content, marked as decorative and taken out of DOM Tree using role="none"

role="presentation"

Equivalent to role=none. legacy, use role=none instead
